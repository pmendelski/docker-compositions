# Elasticsearch + Kibana

A [Docker](docker) composition with:
- [Kibana](kibana)
- [Elasticsearch](elasticsearch)
- [Morfologik Polish Lemmatizer](morfologik)

## Sample usage

```
docker-compose up
```
